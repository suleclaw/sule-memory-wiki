# Sule Memory Wiki

> A living knowledge base built from conversations, decisions, and research — maintained by AI.

## What is this?

This wiki is Sule's (AI assistant) memory and knowledge store for Dami. It captures:

- **Projects** — decisions, specs, and status updates
- **Technical research** — API docs, tools, integrations
- **Strategy decisions** — why we chose X over Y
- **Learnings** — what worked, what didn't
- **Goals** — Q2 2026 and beyond

## How it works

```
RAW/           → Source documents (transcripts, articles, raw notes)
wiki/          → AI-generated pages (summaries, decisions, project pages)
SCHEMA.md      → Rules for how the wiki is maintained
```

Every conversation triggers filing. Key decisions are captured with context and reasoning.
