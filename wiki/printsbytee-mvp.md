# PrintsbyTee MVP

**Owner:** Dami's wife's fashion business
**Live URL:** [printsbytee.co.uk](https://printsbytee.co.uk)
**Repo:** `~/projects/printsbytee`
**Tech Stack:** Next.js + TypeScript + Tailwind CSS + shadcn/ui + Framer Motion
**Status:** Shipped (V1) — actively in use

---

## Overview

Afro-Luxe fashion brand. Premium brand positioning, deep emerald/gold palette, Playfair Display typography. The brand identity is strong but the e-commerce implementation is incomplete — no real product photos, no cart/checkout, manual enquiry flow only.

---

## What Was Built (MVP)

- Product catalogue with filterable grid
- Product detail pages with size selector
- Bento grid homepage layout
- WhatsApp enquiry integration
- Newsletter signup (commented out)
- Mobile-responsive design
- Framer Motion animations throughout

---

## Current Gap

The site now has **real product photography** for 7 of 8 products (Lora Set — Red still has gradient placeholder). Checkout is `mailto:` enquiry links. This is the #1 revenue blocker remaining.

Real product catalogue (PR #9 merged 2026-04-10):
- 5x Lora Set variants (Turquoise, Green, Red & Black, Navy Blue, Red*)
- 1x Aso Oke Kimono Set
- 1x Fringe Bubu
- 1x Naya Jump Suite
- Prices: £30–£40 (down from £185+ mock prices)

See: [[printsbytee-website-review-2026-04-08]] for full issue list (20 issues, checkout is critical).
