# GetRoleUp

> Structured learning platform covering 26 career tracks with interactive terminal, curated YouTube videos, AI tutor, and progress tracking.

## Overview

**Problem:** Tutorial hell — you can watch YouTube forever and never actually grow. GetRoleUp provides the structured path that forces active skill-building, not passive watching.

**Target:** Anyone leveling up or switching careers

**Desired outcome:** Level up or switch careers with confidence

**Key differentiator:** Defined path vs scattered tutorials
