# Obsidian Setup

Dami's personal note-taking and knowledge management tool.

---

## Purpose

Obsidian is used as a local vault for notes, project documentation, and the UI Catalogue reference library.

---

## Wiki Sync

The Sule Memory Wiki (`/root/sule-memory-wiki/`) is synced with an Obsidian vault. Obsidian provides the local editing experience while GitHub Pages hosts the public wiki.

---

## Vault Structure

- `1.Projects/` — active project notes
- `2.Resources/` — reference libraries (UI Catalogue, etc.)
- `3.Resources/` — additional resource categories

---

## Obsidian Plugins Used

- **Templater** — for consistent note templates
- **Dataview** — for querying across notes
- **Git** — for version control syncing
