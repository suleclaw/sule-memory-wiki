## Tech Stack

- **Engine:** Phaser 3
- **Build:** No bundler, no build step — pure HTML/JS
- **Deployment:** Vercel (static hosting)

## Overview

Simple 2D platformer designed to run in a browser. The target use case is an Amazon Fire tablet connected to a keyboard — so controls are keyboard-only, and the game is tuned for that input method.

## History

- **2026-04-07:** Project shipped. Dami's kids are the end users.
- Built as a quick, fun project separate from the agency/client work.

## Notes
